# 前端工程化

## 1. 构建工具
- **Webpack**: 模块打包器，功能强大但配置复杂，构建速度较慢。
- **Vite**: 基于 ESM 的开发服务器，秒级启动，生产环境使用 Rollup 打包。
- **Rollup**: 专注于库的打包，生成的代码更简洁。

## 2. 包管理工具
- **npm**: 嵌套依赖，node_modules 体积大。
- **pnpm**: 软链接 + 硬链接，节省磁盘空间，安装速度快。
- **yarn**: 并行安装，离线缓存。

## 3. 代码规范
- **ESLint**: 检查 JS 代码质量和风格。
- **Prettier**: 代码格式化工具。
- **Husky + Lint-staged**: Git 提交前自动检查。
