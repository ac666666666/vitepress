# TypeScript 基础类型

[TypeScript 官方文档](https://www.typescriptlang.org/zh/)

- `boolean`, `number`, `string`
- `array`, `tuple`, `enum`
- `any`, `void`, `null`, `undefined`
- `never`, `object`

```typescript
let isDone: boolean = false;
let decimal: number = 6;
let color: string = "blue";
let list: number[] = [1, 2, 3];
```
