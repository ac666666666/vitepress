# 数据结构与算法

## 1. 常见数据结构
- **数组 (Array)**: 连续内存，查询快，增删慢。
- **链表 (Linked List)**: 离散内存，增删快，查询慢。
- **栈 (Stack)**: 先入后出 (LIFO)。
- **队列 (Queue)**: 先入先出 (FIFO)。
- **树 (Tree)**: 二叉树, AVL, 红黑树。

## 2. 常见算法
- **排序**: 快速排序 (Quick Sort), 归并排序 (Merge Sort), 冒泡排序。
- **查找**: 二分查找 (Binary Search)。
- **遍历**: 深度优先 (DFS), 广度优先 (BFS)。
